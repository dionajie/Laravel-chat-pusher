# Laravel Chat Pusher

### Description
Chat app using Laravel Framework and Pusher. Just clone and setup a few things.

### How To
- Clone the repo
- Composer install
- Setup .env file (e.g Database and Pusher env)
```sh
PUSHER_KEY=
PUSHER_SECRET=
PUSHER_APP_ID=
```
- Migrate Database
```sh
php artisan migrate
```
